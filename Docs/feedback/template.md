# 基盤リポジトリ改善フィードバック

## 使い方

1. 開発モードで作業中に改善点を見つけたら、このテンプレートをコピーして新しいファイルを作成
2. 改善点を記録
3. プロジェクト完了後、計画モードでこのフィードバックを参照して改善を実施
