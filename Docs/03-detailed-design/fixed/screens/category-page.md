# 画面詳細設計: カテゴリページ

## 基本情報

- **画面名**: カテゴリページ
- **画面ID**: CATEGORY
- **URL**: `/category/[slug]`
- **説明**: カテゴリ別の記事一覧を表示するページ
- **対象ユーザー**: すべてのユーザー

## 画面の目的

指定されたカテゴリに属する記事を一覧表示する。

## レイアウト

トップページと同様のレイアウト。カテゴリ名を表示。

## 画面要素

- カテゴリ名（見出し）
- 記事一覧（トップページと同様）
- ページネーション
