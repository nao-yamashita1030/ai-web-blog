# テスト記事の作成方法

## 方法1: microCMS管理画面から手動で追加（推奨）

### 手順

1. **microCMSの管理画面にログイン**
   - https://[サービスドメイン].microcms.io にアクセス
   - ログイン

2. **コンテンツ一覧を開く**
   - 左側のメニューから「blogs」（または「ブログ」）を選択

3. **新規記事を作成**
   - 右上の「+ 追加」ボタンをクリック
   - 以下の情報を入力：

## 方法2: 書き込み用APIキーを使用（上級者向け）

書き込み用のAPIキーを取得すれば、スクリプトから記事を作成できます。

1. microCMSの管理画面で「設定」→「API設定」を開く
2. 書き込み用のAPIキーを生成
3. `.env.local`に`MICROCMS_WRITE_API_KEY`として設定
4. `npm run create:test-posts`を実行

## 注意事項

- 通常のAPIキーは読み取り専用です
- 記事の作成は管理画面から行うのが最も簡単です
- テスト記事を作成後、開発サーバーを再起動すると記事が表示されます
