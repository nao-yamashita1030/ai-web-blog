# 画面詳細設計: 記事詳細ページ

## 基本情報

- **画面名**: 記事詳細ページ
- **画面ID**: ARTICLE_DETAIL
- **URL**: `/posts/[id]`
- **説明**: 記事の詳細を表示するページ
- **対象ユーザー**: すべてのユーザー

## 画面の目的

記事の全文を表示し、ユーザーが記事を読めるようにする。

## レイアウト

### 画面構成

```
┌─────────────────────────┐
│       ヘッダー           │
├─────────────────────────┤
│                         │
│   記事詳細エリア         │
│  - タイトル              │
│  - 公開日時              │
│  - カテゴリ・タグ        │
│  - アイキャッチ画像      │
│  - 記事本文（マークダウン）│
│  - ソーシャルシェアボタン│
│                         │
│   関連記事エリア（オプション）│
│  - 関連記事一覧          │
│                         │
├─────────────────────────┤
│       フッター           │
└─────────────────────────┘
```

## 画面要素

### 記事タイトル

- **要素名**: タイトル
- **種類**: 見出し（h1）
- **位置**: 記事詳細エリアの上部
- **説明**: 記事のタイトル
- **必須/任意**: 必須

### 記事本文

- **要素名**: 本文
- **種類**: マークダウンコンテンツ
- **位置**: 記事詳細エリア
- **説明**: マークダウン形式の記事本文をHTMLに変換して表示
- **必須/任意**: 必須

### ソーシャルシェアボタン

- **要素名**: シェアボタン
- **種類**: ボタン（複数）
- **位置**: 記事本文の下部
- **説明**: Twitter、Facebook、LINE等へのシェアボタン
- **必須/任意**: 推奨

## エラーハンドリング

### エラーメッセージ

| エラーコード | エラーメッセージ | 表示位置 |
|------------|----------------|---------|
| R002 | 記事が見つかりませんでした | 画面中央 |

## 画面遷移

### 遷移元

- [トップページ](./top-page.md) → この画面
- [カテゴリページ](./category-page.md) → この画面
- [タグページ](./tag-page.md) → この画面
- [検索結果ページ](./search-result-page.md) → この画面

### 遷移先

- この画面 → [トップページ](./top-page.md)（ヘッダーのロゴクリック時）
- この画面 → [カテゴリページ](./category-page.md)（カテゴリクリック時）

## 処理フロー

1. 画面表示時
   - ビルド時に生成された静的HTMLを配信
   - 記事データを取得（ビルド時に含まれている）
   - マークダウンをHTMLに変換
   - 記事詳細を表示

## レスポンシブデザイン

- デスクトップ: 最大幅を設定して読みやすく表示
- タブレット・スマートフォン: 全幅表示

## アクセシビリティ

- 適切な見出し構造
- 画像にalt属性を設定
- キーボード操作に対応
