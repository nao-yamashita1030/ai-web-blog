## ファイル命名規則

APIファイルの命名規則：
- エンドポイントベース: `users.md`, `products.md`
- 機能ベース: `user-authentication.md`, `product-search.md`
- リソースベース: `user.md`, `product.md`

推奨: 機能やリソースを表す分かりやすい名前を使用してください。
