# ルール一覧

このディレクトリには、AI駆動開発基盤リポジトリで使用する各種ルールを管理します。

## ルール一覧

### コミットメッセージのルール

- **ファイル**: [commit-message.md](./commit-message.md)
- **説明**: コミットメッセージの書き方についてのルール
- **内容**: 基本形式、タイプ、件名、本文、モード別の推奨形式など

### フェーズ移行時のルール

- **ファイル**: [phase-transition.md](./phase-transition.md)
- **説明**: 開発モードで次のフェーズに移行する際のコミット・プッシュルール
- **内容**: フェーズ移行のタイミング、コミット・プッシュの手順、コミットメッセージの形式など

## ルールの追加

新しいルールを追加する場合は、以下の手順に従ってください：

1. `rules/` ディレクトリに新しいルールファイルを作成（例: `rules/new-rule.md`）
2. この `README.md` に新しいルールを追加
3. 必要に応じて `.cursorrules` や `README.md` に参照を追加
