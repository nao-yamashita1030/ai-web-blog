# 画面詳細設計一覧

## 概要

このディレクトリには、ブログサイトの各画面の詳細設計を管理します。

## 画面一覧

### 主要画面

1. [トップページ（記事一覧）](./top-page.md)
   - URL: `/`
   - 説明: 最新記事の一覧を表示

2. [記事詳細ページ](./article-detail-page.md)
   - URL: `/posts/[id]`
   - 説明: 記事の詳細を表示

3. [カテゴリページ](./category-page.md)
   - URL: `/category/[slug]`
   - 説明: カテゴリ別の記事一覧を表示

4. [タグページ](./tag-page.md)
   - URL: `/tag/[slug]`
   - 説明: タグ別の記事一覧を表示

5. [検索結果ページ](./search-result-page.md)
   - URL: `/search?q={query}`
   - 説明: 検索結果の一覧を表示

### 共通コンポーネント

- ヘッダー
- フッター
- ナビゲーション

## 画面遷移

```
[トップページ]
    ├─→ [記事詳細ページ]
    ├─→ [カテゴリページ] ─→ [記事詳細ページ]
    ├─→ [タグページ] ─→ [記事詳細ページ]
    └─→ [検索結果ページ] ─→ [記事詳細ページ]
```

## デザイン方針

- モダン・トレンディなデザイン
- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- アクセシビリティ対応（WCAG 2.1 Level AAを目標）
